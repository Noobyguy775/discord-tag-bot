# Privacy Policy
This privacy policy details what information **unnamed discord tag bot** (also referred to as "the bot") collects, stores, and uses to provide services to any user who interacts with the bot.

## I. Data collected
This data is essential for the bot to function.
- Discord User info, such as:
   - In a guild, highest role color & nickname
   - UserID.
- Message info, such as:
   - Message content.
- Interaction data, such as:
   - Parameters sent to the bot.

Any messages that do not trigger functionality of the bot are not logged or stored.

## II. Data Security
User data is never stored for longer than it is needed. Data is deleted when no longer needed.

## III. Information collected
The bot collects usage data on which tags are used most often. This data is anonymous and not linked to a user.

All functionality of this bot can be found in the repository that this policy is written in. 

## IV. Cessation/Deletion
If you wish to discontinue use of the bot and therefore the use and storage of your data, please use the contact below. Note that cessation will result in a block from any of the bot's current and future functionality.
